第二阶段：创建你的第一个仓库（Repository）（5–10分钟）
目标：拥有一个属于自己的“网盘项目文件夹”
1.  登录后，点击右上角 + → New repository
2.  填写信息（非常重要，第一次建议这样填）：
	•  Repository name（仓库名）：my-first-repo 或 hello-github（随便起，英文小写+中划线推荐）
	•  Description（描述）：可以写「我的第一个GitHub项目」
	•  Public / Private：选 Public（公开的，新手建议先公开）
	•  勾选 → Add a README file（非常重要！）
	•  可选：Add .gitignore → 选你主要语言（Python / Node / Java 等）或先不选
	•  License：新手先选 MIT License（最宽松）或不选
3.  最下面绿色按钮 Create repository
→ 成功！你现在有了一个仓库！
第三阶段：三种最常用玩法（新手最该先会的）
玩法A：直接在网页上写代码/改文档（最简单，零安装）
适合：写文档、记笔记、写博客、改README
1.  进入你的仓库
2.  点右上角 Add file → Create new file
3.  文件名输入：notes.md（.md是Markdown格式）
4.  写点内容，例如：


好的！下面我用最简单、最保姆级的方式，一步一步带你从零开始学会用 GitHub（2026年最新流程基本没变）。
我们分成几个阶段，你可以按顺序来，建议边看边操作，第一次可能花30–60分钟，之后就很顺手了。
第一阶段：注册 GitHub 账号（5分钟）
1.  打开浏览器，进入： https://github.com
2.  右上角点击 Sign up（注册）
3.  按顺序填写：
	•  邮箱（建议用你常用的 Gmail/Outlook/公司邮箱）
	•  密码（建议：字母+数字+符号，至少12位以上）
	•  用户名（Username）：这是你的GitHub ID，全网唯一，建议起一个好记、专业的名字（以后简历会写上去）
	•  验证码（人机验证）
4.  选择免费的 Free 计划 → Continue
5.  回答几个问题（随便选，跳过也行）→ Create account
6.  去邮箱收验证邮件，点链接完成验证
→ 恭喜！你已经有GitHub账号了！
